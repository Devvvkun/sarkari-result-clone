# 📰 Sarkari Result Clone

## 🔍 Description
This project is a frontend clone of the Sarkari Result website built using HTML and CSS.  
It replicates the layout of a government job portal including sections like Results, Admit Cards, Latest Jobs, and more.

This project helped me understand real-world website structure and layout design.

---

## 🚀 Features
- Header with logo and title
- Navigation bar with multiple sections
- Latest news ticker using marquee
- Highlighted important links (Apply Online, Results, etc.)
- Multiple content cards:
  - Results
  - Admit Card
  - Latest Jobs
  - Answer Key
  - Syllabus
  - Admission
- Structured layout using CSS

## ⚠️ Limitations
- Not responsive (desktop only)
- Uses `<marquee>` (deprecated in modern HTML)
- Static content (no backend or dynamic data)
